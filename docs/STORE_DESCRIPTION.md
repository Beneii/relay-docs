# App Store / Play Store Copy

## App Name
Relay — Dashboard Launcher

## Subtitle (iOS) / Short Description (Android)
Save your dashboards. Get push alerts. One tap to open.

## Description

Relay is a tiny mobile launcher for your dashboards, tools, and automations.

Save any web-based dashboard as an app. Open it instantly with one tap. Receive native push notifications from your systems via a simple webhook — and tap the alert to jump straight to the right dashboard.

**What Relay does:**
- Save dashboards, admin panels, and tools as apps
- Launch them full-screen with a single tap
- Receive push notifications from any system via webhook
- Tap a notification to open the exact dashboard it came from

**Built for:**
- Self-hosted tools (Grafana, Home Assistant, n8n, etc.)
- Internal dashboards and admin panels
- CI/CD pipelines and build monitors
- Automation and bot control panels
- Any web-based system you check regularly

**How notifications work:**
Each saved app gets a unique webhook URL. Point your systems at it — Relay delivers native push notifications to your phone. Tap the alert to open the right dashboard instantly.

No accounts to manage. No complex setup. Just save, launch, and get alerts.
